# TablelandSample

Tableland を学習するためのリポジトリです。

### SDK のインストール

```bash
npm install --save @tableland/sdk
```

### テーブルにつける名前のフォーマット

```txt
{prefix}_{chainId}_{tableId}.
```

### クエリで実際に取得した結果の例

```bash
[
  { id: 0, val: 'Bobby Tables!' },
  { id: 1, val: 'Bobby Tables!' },
  { id: 2, val: 'Bobby Tables!!' },
  { id: 3, val: 'Bobby Tables!' },
  { id: 4, val: 'Bobby Tables!!' }
]
```
